# Crypto Gas Fee Checker

This project checks gas fees.
It shows how much a transaction may cost.
You can learn how gas works on Ethereum.

This project is made for beginners.
All code is simple and clear.

## Features
- Check current gas price
- Show slow, normal, fast fee
- Ethereum testnet support
- Command line tool

## Requirements
- Node.js installed
- RPC provider account
- Internet connection

## Setup
1. Clone this repo
2. Open folder in terminal
3. Run: npm install
4. Create a file called .env
5. Add your RPC URL
6. Run: node gas.js

## Example
Run the script.
Gas prices will show in terminal.

## Learning Goals
- Learn gas basics
- Learn network fees
